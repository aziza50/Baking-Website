# Magnolia Bakery Website

Designed and built an engaging website crafted for a small bakery to display its menu and allow customers to place orders online.

---

## Project Overview

Website purpose:

- Display the bakery’s menu in a clean and inviting layout
- Add a portfolio page for the bakery to showcase their work
- Provide an ordering form for customers
- Future plan: integrate a payment transaction system as the bakery grows

---

## Tech Stack

- **React JS** – for building dynamic, interactive UI
- **Tailwind CSS** – for efficient styling
- **React-Motion** – animations
- **AWS S3 Bucket** – for storing and serving images

---

## Future Improvements

- Add secure payment processing
- Track order history
- Customer reviews
- Mobile friendly Design (In-process)
